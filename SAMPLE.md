# Sample game

## Notation

All reachable cells are numbered 1 to 32. Numbering starts with Black's double corner.

### Move notation
```
<from>-<to>
5-9
```

### Capture notation
```
<from>x<to>
5x14
```

### Game result
```
BW - Black wins
WW - White wins
```
